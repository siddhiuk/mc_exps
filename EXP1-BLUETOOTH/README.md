# MC-EXP-01-BLUETOOTH

## Aim
To develop an Android application that can enable Bluetooth, discover devices, connect, and exchange simple data.

## Objectives
* Learn Bluetooth API in Android
* Pair and connect Bluetooth devices
* Transfer data between paired devices

## Software Required
* Android Studio
* Java / Kotlin SDK

## Procedure
1. Create new Android Studio project
2. Add Bluetooth permissions
3. Enable Bluetooth Adapter
4. Discover nearby devices
5. Establish connection (RFCOMM)
6. Send/Receive simple text
